Detalles del Cuestionario de Búsqueda de Hoteles

Nombre Usuario:

Se emplea el input type="text", debido a que el usuario debe escribir su nombre y necesita un campo de texto de línea simple. En este campo se requiere de un nombre de usuario, que permita identificar ante el sistema a la persona que va a realizar el viaje, y va a ser responsable de la(s) habitación(es).

Contraseña:

Se emplea el input type="password", debido a que se necesita ingresar una contraseña, y este atributo tiene un control de línea simple cuyo valor permanece oculto. Se requiere ingresar una contraseña, que permaecerá secreta y que ayudará a verificar que la identidad del usuario sea correcta.

Repetir Contraseña:

Se emplea el input type="password", debido a que se necesita ingresar nuevamente una contraseña, y este atributo tiene un control de lìnea simple cuyo valor permanece oculto.Se necesita que se ingrese de nuevo la contraseña para verificar la escritura correcta de la misma en el apartado anterior.

Dirección:

Se emplea el input type="text", debido a que el usuario debe escribir su nombre y necesita un campo de texto de línea simple. Es necesario que se escriba la dirección del usuario por datos de seguridad del hotel y del usuario.

Ciudad:

Se emplea el input type="text", debido a que el usuario debe escribir su nombre y necesita un campo de texto de línea simple. Se requiere que el usuario ingrese el nombre de la ciudad a la que va a viajar.

Provincia:

Se emplea el input type="text", debido a que el usuario debe escribir su nombre y necesita un campo de texto de línea simple. Se requiere que el usuario ingrese el nombre de la provincia a la que va a viajar.

Número de Huéspedes:

Se emplea el input type="number", debido a que el usuario debe escribir el número de huéspedes, y este atributo es un control para introducir números.Se requiere saber el nùmero de huéspedes para saber el número de personas que van a viajar y también para realizar un control de la capacidad del hotel.

Servicios:

Se emplea el input type="checkbox", ya que se necesitan casillas de selección. Se presenta una lista de servicios para cubrir las necesidades y requerimientos de los huéspedes, para una mejor experiencia.

Tipo de Habitación:

Se emplea el input type="radio", ya que se necesita marcar un único ítem de una lista de opciones. Se pone a disposición una variedad de tipos de habitaciones para comodida de los huéspedes, pueden variar sus precios por varios factores (como paisaje, concurrencia, etc).

Pensión:

Se emplea el input type="radio", ya que se necesita marcar un único ítem de una lista de opciones. El tiempo que los huéspedes ocupen una habitación está relacionada directamente con la tarifa.

Comentarios y Sugerencias:

Se usaa el atributo "textarea", debido a que se necesita escribir una cantidad de texto sin límites. Aquí puede expresarse de las experiencias buenas y malas que hayas sucedido durante su estadía. También se puede escribir recomendaciones que ayuden al hotel a mejorar su atención y/o servicio.

E-Mail:

Se emplea el input type="text", debido a que el usuario debe escribir su nombre y necesita un campo de texto de línea simple. Es importante que se escriba el correo electrónico (e-mail) del usuario, para la transferencia de información con el hotel.

Política de Privacidad:

Se emplea el input type="checkbox", ya que se necesita una casilla de selección. Es necesario que se lea y acepte la política de privacidad para evitar inconvenientes legales de cualquier tipo. Si no se acepta, no se podrá registrar.

Limpiar:

Se emplea el input type="reset", porque es un botón que restaura el contenido del formulario. Esta opción nos permite borrar los campos que se llenaron anteriormente.

Enviar:

Se emplea el input type="submit", porque este botón envía el formulario. Esta opción nos permite enviar la informacion llenada en los campos anteriores, hacia un correo determinado.
